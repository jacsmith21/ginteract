# Git Checkout
Quickly checkout your most recent branch!

## Installation
Requirements: Python 3.*
```
pip install gitcheckout
git config --global alias.ch '!gitcheckout'
```

## Usage
```
git ch

# or use the command directly
gitcheckout
```
